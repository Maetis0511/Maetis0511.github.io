git remote

Utilité

git remote gère un ensemble de dépôts suivis.

Commande

    
git remote add [-t (branch)] [-m (master)] [-f] [--[no-]tags] [--mirror=(fetch|push)] (name) (url)
git remote renom (old) (new)
git remote remove (name)
git remote [-v | --verbose] show [-n] (name)…​

Description

La commande git remote permet de gérer un ensemble de dépôts (« distants ») dont vous suivez les branches.

Options

  • add

    Ajoute un distant nommé (name) pour le dépôt à (url).

  • rename

    Renomme le distant nommé (old) en (new). Toutes les branches de suivi à distance et les paramètres de configuration du distant sont mis à jour.

  • remove
    rm

    Supprime le distant nommé (name). Toutes les branches de suivi à distance et tous les paramètres de configuration du distant sont supprimés.

  • show

    Fourni quelques informations sur le dépôt (name).
