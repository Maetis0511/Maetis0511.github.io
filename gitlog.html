git log

Utilité

git log affiche la liste de tous les commits dans l'ordre chronologique.

Commande

    
git log [(options)] [(revision-range)] [[--] (path)…​]

Description

La commande git log permet l'affichage de la liste de tous les commits dans l'ordre chronologique.
