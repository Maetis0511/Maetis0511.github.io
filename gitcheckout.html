git checkout (branch)

Utilité

git checkout (branch) permet de basculer sur une autre branche ou restaurer des fichiers de l’arbre de travail.

Commande

    
git checkout [-q] [-f] [-m] [-b] [(branch)]

Description

La commande git checkout (branch) permet de se préparer à travailler sur (branch), basculer dessus en mettant l’index et les fichiers de l’arbre de travailà jour, et en pointant HEAD sur cette branche. Les modifications locales aux fichiers de l’arbre de travail sont conservées, de sorte qu’elles peuvent être validées sur la (branch).

Options

  • -q
    --quiet

    Supprime les messages d’état.

  • -b (newbranch)

    Crée une nouvelle branche nommée (newbranch).
